Description

The aim of playing Jotto is guessing a secret word. Classically, a word in Jotto can only contain five letters, but in this variation of the game the length of a word, which is always communicated to the user, can change depending on the random word chosen from the database.

At each step, the user is given the chance to submit a guess to the secret word. The length of the guess and that of the hidden word must match, otherwise an error message notifies the user that they have submitted an invalid guess.

When a guess to the secret word satisfies all the validity conditions (namely, same length) the screen will display the number of common letters shared by the two words, and the count of equal letters with equal position inside both words. By combining the information from both fields, the user can infer what word is to be guessed and be informed when they win the game.

When the guessed word is correct, the program (more exactly, the server) will reply with a common letters and equal letters with equal position fields equal between them and the length of the secret word (already known beforehand).

The Jotto Wikipedia article describes this logic word game more carefully.
